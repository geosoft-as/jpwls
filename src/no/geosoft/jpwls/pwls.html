GeoSoft - REST API (PoC) for the OSDU Practical Well Log Standard (PWLS)

Consume

Access PWLS from any software system using URL queries like in the examples below:

Properties

  • http://13.60.27.155/properties
  • http://13.60.27.155/properties?name=depth
  • http://13.60.27.155/properties?quantity=area

  • http://13.60.27.155/companies?companyCode=440

  • http://13.60.27.155/loggingMethods

Tool classes
